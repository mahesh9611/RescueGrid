# RescueGrid Website

A professional static website prototype for the RescueGrid emergency hospital and ambulance tracking platform. The site highlights resource availability, live ambulance tracking, emergency alerts, and an investor-ready pitch.

## Features
- Interactive resource dashboard (beds, ICU, oxygen, blood)
- Emergency alert button with real-time notifications
- Live ambulance tracking with animated route progress
- Professional hospital resource table
- **Patient App Demo**: Location-based hospital search, voice search, live doctor availability, SMS/WhatsApp alerts
- Investor pitch section with key statistics
- Responsive design for all devices

## Patient App Features
- **Location-based hospital list**: Shows nearest hospitals first
- **Fuzzy search**: Type-ahead search with intelligent matching
- **Voice search**: Mic button for hands-free hospital search
- **Live doctor status**: Real-time availability with timestamps
- **Multi-channel alerts**: Send emergency details via SMS or WhatsApp
- **Emergency categories**: Pre-defined options (heart attack, stroke, accident, etc.)

## Files
- `index.html` — homepage layout and content
- `styles.css` — responsive design and visual styling
- `script.js` — interactive features for resource tracking and ambulance updates

## Usage
Open `index.html` in a browser to view the website.

## Adding Images
To make the site more professional, replace placeholder images with actual ones:
1. Add `emergency-response.jpg` or similar in the website folder
2. Update the `src` in `index.html` hero section
